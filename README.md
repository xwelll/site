# Тренажер подготовки к экзамену

## Что реализовано
- 2 режима:
  - `Все вопросы`: полная перемешка с обязательным чередованием по блокам и разделам.
  - `Билет 15`: 1 случайный вопрос из каждой комбинации `блок + раздел` (5×3).
- Улучшенная статистика:
  - попытки, правильные, точность;
  - покрытие базы (`решено / всего`);
  - текущая и лучшая серия;
  - таблица точности по блокам и разделам;
  - список слабых мест (вопросы с наибольшим числом ошибок).
- Устойчивая проверка ответов:
  - одиночные и множественные варианты (`А`, `А,Б,Г`);
  - соответствия и последовательности (`1-А, 2-Б` или `Г,Б,А`);
  - поддержка смешанных/битых буквенных форматов при нормализации.

## Запуск
```bash
py -3 -m http.server 8000
```

Открыть: `http://localhost:8000`

## Проверка качества банка вопросов
```bash
py -3 scripts/validate_questions.py
```

## Пересборка вопросов из DOCX
```bash
py -3 scripts/extract_questions.py
```
